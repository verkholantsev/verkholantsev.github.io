# verkholantsev.ru

This is [verkholantsev.ru](https://verkholantsev.ru) repo.

## How to build

```shell
# clone repo
git clone git@github.com:verkholantsev/verkholantsev.github.io.git

# change directory
cd verkholantsev.github.io

# install dependencies
npm install

# run gulp
./node_modules/.bin/gulp
```
